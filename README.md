# ProyectoHardware
Programación del juego Sudoku en una placa 'Embest S3CEV40' con diferentes funcionalidades extra
- Cálculo de candidatos para cada celda
- Control de errores por celdas
- Interfaz gráfica en pantalla LCD
- Interacción táctil
- ...
